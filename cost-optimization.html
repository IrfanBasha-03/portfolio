AWS Cloud Cost Optimization

This Lambda-based automation script finds and deletes unused EBS snapshots not associated with active EC2 volumes — optimizing storage and cost in cloud environments.

Technologies Used

  • AWS Lambda & IAM
  • Boto3 (Python SDK)
  • EC2, EBS Snapshots

Challenges Faced

  • Accurately identifying stale snapshots without disrupting running services.
  • Handling permission boundaries using IAM roles.
  • Ensuring secure deletion logging with CloudWatch.

Sample Code Snippet


import boto3

ec2 = boto3.client('ec2')
snapshots = ec2.describe_snapshots(OwnerIds=['self'])['Snapshots']

for snap in snapshots:
    if "automated" not in snap['Description']:
        ec2.delete_snapshot(SnapshotId=snap['SnapshotId'])
        print("Deleted:", snap['SnapshotId'])
